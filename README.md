# Jogo-da-Velha
Projeto de Jogo da Velha desenvolvido utilizando HTML, CSS e JavaScript, com foco em lógica de programação, interatividade e manipulação do DOM. 
O jogo oferece duas formas de jogar:  
👥 Modo 2 jogadores (jogador vs jogador) 
🤖 Modo contra IA, onde o usuário pode desafiar uma lógica automatizada que toma decisões durante a partida
